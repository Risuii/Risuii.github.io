Proyek Unggulan

Portofolio teknis dengan konteks bisnis dan kontribusi nyata.

QRIS CPM

Layanan backend untuk modul pembayaran QRIS (Quick Response Indonesian Standard) dalam skema CPM (Customer-Presented Mode)—merchant menerima pembayaran via QR yang dipindai pelanggan.

Peran & kontribusi

  • Merancang dan mengimplementasikan REST API dengan validasi input ketat
  • Membangun layer autentikasi & otorisasi akses layanan
  • Mengelola skema database (migrasi, ORM) agar transaksi dan master data konsisten
  • Memastikan endpoint siap diintegrasikan tim frontend / channel pembayaran

Hasil: API terstruktur, aman, dan mudah di-maintain untuk alur pembayaran QRIS pada lingkungan perbankan/fintech.

Go Gin REST API PostgreSQL GORM

Visa Authentication Service

Layanan autentikasi untuk transaksi Visa—memverifikasi dan mengamankan request sebelum diproses ke core banking/payment, termasuk penanganan payload sensitif.

Peran & kontribusi

  • Mengembangkan service autentikasi end-to-end (Go)
  • Menerapkan enkripsi/tokenisasi payload dengan JWE
  • Validasi kontrak request/response dan error handling yang konsisten
  • Berkolaborasi dengan tim integrasi agar service siap production

Hasil: Jalur autentikasi transaksi Visa lebih aman, terstandar, dan siap skala untuk kebutuhan internal bank.

Go Gin REST API JWE PostgreSQL GORM
